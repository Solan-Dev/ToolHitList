## Template

Create new ADRs as `ADR-NNN-short-title.md` using this format:

```markdown

## Status
Proposed / Accepted / Superseded

## Context
What situation prompted this decision?

## Decision
What was decided?

## Consequences
What are the trade-offs?
```
